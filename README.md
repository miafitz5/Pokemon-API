# Pokemon-API

A simple API for retrieving information about Pokémon. This project is written in Python and provides endpoints to access various Pokémon data.

## Features
- Retrieve Pokémon details
- Sprites
- List all available Pokémon
- Search for Pokémon by name or type

## Getting Started

### Prerequisites
- Python 3.7+
- pip

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/miafitz5/Pokemon-API.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Pokemon-API
   ```
3. (Optional) Create and activate a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
4. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

To run the API server:
```bash
python Pokemon.py
```
